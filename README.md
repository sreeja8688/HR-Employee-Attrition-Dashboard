# HR Employee Attrition Analysis Dashboard

## Overview
This project presents an interactive HR Employee Attrition dashboard developed using Microsoft Power BI. The dashboard helps analyze employee attrition and identify factors contributing to employees leaving the organization.

## Features
- Total Employees KPI
- Employees Left KPI
- Attrition Rate KPI
- Employee Attrition Pie Chart
- Attrition by Department
- Attrition by Age Group
- Attrition by Overtime
- Interactive slicers for Department, Age Group, and Business Travel

## Tools Used
- Microsoft Power BI
- IBM HR Analytics Employee Attrition Dataset
